Connecting to the Lab
=====================

Start your session by opening the following `link <https://dcloud.cisco.com/expo/b3gi3rj4fngbp2k39h3o55mlm>`__

Connect to AnyConnect (Optional)
--------------------------------

Use the ``Cisco AnyConnect Secure Mobility Client`` to connect through a VPN, information will be provided after registering via above link in the :guilabel:`Details` tab.

.. image:: images/expo-anyconnect.png
    :width: 75%
    :align: center

With the VPN connected, launch your preferred RDP Client to connect to the server specified in the next step.

If you cannot use a VPN, you can launch the Remote link from the :guilabel:`Network` tab.

.. image:: images/expo-remote-desktop.png
    :width: 75%
    :align: center


Once inside the remote desktop connection, you can acccess network devices, Docker, GitHub repo and documentation.
In addition you can create your enviroment to start to build your self service portal.
